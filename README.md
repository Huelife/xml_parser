# xml_parser
Simple XML parser using ElementTree
---------------------------------
1) Program parses a file for city names.
2) Download all files.
3) Run 'xml_parser.py'.
4) Program will parse 'dataMay-24-2019.xml'.
5) Program will also write the data to 'information.txt'.
